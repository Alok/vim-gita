#
# STATUS MAPPING
#
# <C-l> : Update status
# cc    : Switch the buffer to a commit buffer
# cC    : Switch the buffer to a new commit buffer
# cA    : Switch the buffer to a new commit buffer in AMEND mode
# 
# e     : Open a selected file
# E     : Open a selected file via vsplit
# d     : Open a diff between a current and a specified version of a selected file
# D     : Open a current and a specified version of a selected file in diff mode
# s     : Open a LOCAL and REMOTE of a selected file (for conflicted)
# S     : Open a BASE, LOCAL and REMOTE of a selected file (for conflicted)
#
# <<    : Stage changes of a selected file
# >>    : Unstage changes of a selected file
# --    : Stage/Unstage (Toggle) changes of a selected file
# ==    : Discard changes of a selected file on working tree
#
# -a    : Add a selected file
# -r    : Reset a selected file
# -d    : Remove a selected file
# -c    : Checkout a specified version of a selected file
# -o    : Checkout a ours version of a selected file (for conflicted)
# -t    : Checkout a theirs version of a selected file (for conflicted)
#
